This is my attemt to use next.js to rewrite the wizard.
I decided to do it with usual react and react form library
VS code extensions used:
Material Icon Theme
Prettier
Simple React Snippets

Following instructions here:
https://nextjs.org/docs/app/getting-started/installation

1. create github repo and .gitignore file 
2. $ npm init -y
3. $ npm install next@latest react@latest react-dom@latest
4. add scripts to package.json
{
  "scripts": {
    "dev": "next dev",
    "build": "next build",
    "start": "next start",
    "lint": "next lint"
  }
}

5. project structure:
   $ mkdir app public
   $ touch app/layout.js  app/page.js
   $ mkdir public

6. $ npm run dev   

add jsconfig.json
